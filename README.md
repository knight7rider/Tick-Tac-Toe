🧩 Basic Description

Tic-Tac-Toe is played on a 3×3 grid. Two players take turns marking spaces in the grid:

One player uses X
The other player uses O
🎯 Objective

The goal is to be the first player to get three of your marks in a row:

Horizontally
Vertically
Diagonally
🔄 How the Game Works
The game starts with an empty grid.
Players take turns placing their symbol (X or O) in an empty cell.
Once a cell is filled, it cannot be used again.
The game continues until:
One player wins, OR
All 9 cells are filled (this results in a draw).
🏆 Winning Example

A player wins if they form a pattern like:

X | X | X
O | O |  
  |   |  
🧠 Strategy Element

Even though it looks simple, Tic-Tac-Toe involves strategy:

Blocking your opponent’s moves
Planning ahead
Creating multiple winning chances
💡 Fun Fact

If both players play perfectly, the game will always end in a draw.
